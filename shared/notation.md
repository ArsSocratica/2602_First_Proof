# Shared Notation Reference

## Common Symbols

| Symbol | Meaning |
|--------|---------|
| $\mathbb{R}, \mathbb{C}, \mathbb{Q}, \mathbb{Z}$ | Real, complex, rational, integer numbers |
| $\mathbb{T}^n$ | $n$-dimensional torus |
| $\mathcal{D}'$ | Space of distributions |
| $\mathrm{GL}_n(F)$ | General linear group over field $F$ |
| $N_r$ | Upper-triangular unipotent subgroup of $\mathrm{GL}_r$ |
| $\otimes$ | Tensor/Kronecker product |
| $\odot$ | Khatri-Rao (column-wise Kronecker) product |
| $\text{vec}(\cdot)$ | Vectorization of a matrix (stack columns) |
| $\text{diag}(\cdot)$ | Diagonal matrix |
| $I_r$ | $r \times r$ identity matrix |
| $E_{i,j}$ | Matrix with 1 in $(i,j)$-entry, 0 elsewhere |
| $\boxplus_n$ | Finite free convolution of degree-$n$ polynomials |
| PSD | Positive semidefinite |
