# Problem 6 — Existence of ε-Light Subsets

## Result

**Partial result.** The full conjecture remains open. We establish:

1. **Theorem A** (Independence regime): If $\bar{d} \leq 6/\epsilon - 1$, an ε-light subset $S$ exists with $|S| \geq \epsilon n/6$.
2. **Corollary A.1** (Effective resistance decomposition): For any graph, $\exists$ independent set $I$ with $|I| \geq \epsilon n/3$ and all edges in $G_I$ having $R_{\text{eff}}(e) \leq \epsilon$.
3. **Proposition C** (Upper bound): $c \leq 1/2$ (tight: disjoint cliques with $m \approx 2/\epsilon$).
4. **Theorem E** (Barrier greedy, conditional): If $\Delta_I = O(1)$, then $c = 1/6$ via BSS-style barrier.
5. **Theorem F.1** (Complete graph): For $K_n$, a balanced partition into $k$ groups is $\epsilon$-light with $|S_i| \geq \epsilon n/2$. Proved by direct computation.
6. **Lemma H** (Star norm identity): $\|\tilde{L}_v^*\| = 1$ for all $v$ in all connected graphs. Yields joint bound $\sum_j M_j \preceq \Pi$ and scalar pigeonhole $\min_j w^T M_j w \leq \epsilon/3$.
7. **Proposition G** (Barbell obstruction): The greedy with $k = \lceil 2/\epsilon \rceil$ bins hits $\mu_{\max} = 1$ on barbell $B_m$ with $m = k$. **$c = 1/2$ via partitions is impossible.**
8. **Theorem I** (Degeneracy, **proved**): If degeneracy $d < k = \lceil 3/\epsilon \rceil$, the greedy with degeneracy ordering produces an $\epsilon$-light partition with $c = 1/3$. Covers all planar graphs ($d \leq 5$, $\epsilon < 0.6$), trees, bounded-treewidth, sparse regular graphs.
9. **Theorem F** (Multi-bin barrier, computational): With $k = \lceil 3/\epsilon \rceil$ bins, the greedy achieves $\mu_{\max} \leq 2/3$ across all tested graphs (19 families, $n \leq 100$, $\epsilon \in \{0.1, 0.15, 0.2, 0.3\}$, 500 trials each, 0 stuck). Would give $c = 1/3$.

**Strongest result:** Theorem I **proves** $c = 1/3$ for all graphs with degeneracy $< \lceil 3/\epsilon \rceil$ (including all planar graphs). The log-det multi-bin barrier with $k = \lceil 3/\epsilon \rceil$ bins achieves $c = 1/3$ computationally for all tested graphs. The barbell obstruction (Proposition G) proves $k = \lceil 2/\epsilon \rceil$ is insufficient, killing the $c = 1/2$ target via partitions. The star norm identity (Lemma H) gives the joint bound $\sum_j M_j \preceq \Pi$ and scalar pigeonhole $\min_j w^T M_j w \leq \epsilon/3$. The projector-rank analysis (Session 12b) reveals the empirical mechanism: at the worst greedy step, the best bin has load $= 0$, rank$(\delta) = 1$, and $\mu = R_{\text{eff}}/\epsilon < 1$. The remaining gap: prove this mechanism formally for $k$-core vertices with $\geq k$ back-neighbors.

---

## Problem Statement

For a graph $G = (V, E)$ with $n = |V|$ and Laplacian $L$, the induced subgraph Laplacian is $L_S = \sum_{\{u,v\} \in E(S,S)} L_{uv}$. A set $S \subseteq V$ is **ε-light** if $L_S \preceq \epsilon L$.

**Question (Spielman).** Does there exist $c > 0$ such that for every graph $G$ and every $\epsilon \in (0,1)$, there exists an ε-light subset $S$ with $|S| \geq c\epsilon n$?

---

## Normalized Formulation

Let $\Pi$ denote the orthogonal projector onto $\mathbf{1}^\perp$ and define $\tilde{M} := L^{+/2} M L^{+/2}$. Then $L_S \preceq \epsilon L$ iff $\|\tilde{L}_S\| \leq \epsilon$. The normalized edge Laplacians are rank-1 PSD with $\|\tilde{L}_{uv}\| = R_{\text{eff}}(u,v)$ and $\sum_e \tilde{L}_{uv} = \Pi$.

*Proof.* $\tilde{L}_{uv} = L^{+/2}(\mathbf{e}_u - \mathbf{e}_v)(\mathbf{e}_u - \mathbf{e}_v)^T L^{+/2}$ is rank-1 PSD with trace $R_{\text{eff}}(u,v)$. For rank-1 PSD, $\|\cdot\| = \text{tr}(\cdot)$. Summing: $\sum_e \tilde{L}_{uv} = L^{+/2}LL^{+/2} = \Pi$. $\square$

---

## Lemma 1 (Linearization)

*For any $S \subseteq V$:* $L_S \preceq \frac{1}{2}\sum_{v \in S} L_v^*$.

*Proof.* For $s_u, s_v \in \{0,1\}$: $s_u s_v \leq \frac{1}{2}(s_u + s_v)$. (If either is 0, LHS $= 0 \leq$ RHS; if both are 1, LHS $= 1 =$ RHS.) Since $L_{uv} \succeq 0$:
$$L_S = \sum_e s_u s_v L_{uv} \preceq \sum_e \tfrac{s_u + s_v}{2} L_{uv} = \tfrac{1}{2}\sum_{v \in S} L_v^*. \quad \square$$

This converts the quadratic vertex-edge coupling into a linear sum, at the cost of a factor-of-2 looseness. Tight when $S = V$.

---

## Theorem A (Independence Regime)

*If $\bar{d} \leq 6/\epsilon - 1$, there exists an ε-light $S$ with $|S| \geq \epsilon n/6$.*

*Proof.* Turán gives independent set $|S| \geq n/(\bar{d}+1) \geq \epsilon n/6$. Independent $\Rightarrow$ $L_S = 0$. $\square$

**Remark.** Degenerate: ε-lightness is trivial ($L_S = 0$). Handles sparse graphs only.

---

## Corollary A.1 (Effective Resistance Decomposition)

*For any graph $G$ and $\epsilon \in (0,1)$, $\exists$ independent set $I$ with $|I| \geq \epsilon n/3$ and all edges in $G_I$ satisfying $R_{\text{eff}}(e) \leq \epsilon$.*

*Proof.* Foster's theorem: $\sum_e R_{\text{eff}}(e) = n-1$. So $|E_{\text{hi}}| = |\{e : R_{\text{eff}}(e) > \epsilon\}| < n/\epsilon$. The subgraph $G_{\text{hi}}$ has average degree $< 2/\epsilon$. Turán: independent set $I$ in $G_{\text{hi}}$ with $|I| \geq n/(2/\epsilon + 1) \geq \epsilon n/3$. Since $I$ is independent in $G_{\text{hi}}$, all edges in $G_I$ have $R_{\text{eff}} \leq \epsilon$. $\square$

---

## Corollary A.2 (Trace-Small Case)

*If $\text{tr}(\tilde{L}_I) \leq \epsilon$, then $I$ is ε-light with $|I| \geq \epsilon n/3$.*

*Proof.* $\|\tilde{L}_I\| \leq \text{tr}(\tilde{L}_I) \leq \epsilon$. $\square$

---

## Theorem B (Expectation Bound)

*Sampling each vertex with probability $p$: $\mathbb{E}[\tilde{L}_S] \preceq p\Pi$.*

*Proof.* Lemma 1 gives $\tilde{L}_S \preceq \frac{1}{2}\sum_v \xi_v \tilde{L}_v^*$ pointwise. Taking expectations: $\mathbb{E}[\cdot] \preceq \frac{p}{2}\sum_v \tilde{L}_v^* = p\Pi$. $\square$

**Remark.** True expectation is $\mathbb{E}[\tilde{L}_S] = p^2\Pi$; linearization loses factor $1/p$.

---

## Proposition C (Upper Bound)

*$c \leq 1/2$.*

*Proof.* Fix $\epsilon$ and let $m = \lfloor (2-\delta)/\epsilon \rfloor$ for small $\delta > 0$, so $\epsilon m < 2$. Let $G$ be $n/m$ disjoint copies of $K_m$. If $S$ contains two vertices in a component, $L_{\{u,v\}}$ has eigenvalue 2, but $\epsilon$-lightness requires $2 \leq \epsilon m < 2$, contradiction. So $|S| \leq n/m$ and $|S|/(\epsilon n) \leq 1/(\epsilon m) \to 1/2$. $\square$

---

## Proposition D.1 (Relaxed ε-Lightness)

*For any graph $G$ and $\epsilon \in (0,1)$, $\exists S \subseteq V$ with $|S| \geq \epsilon^2 n/12$ such that:*
$$L_S \preceq C\left(\epsilon\sqrt{\log n} + \Delta_I\epsilon\log n\right) L$$
*where $C > 0$ is absolute and $\Delta_I = \max_{v \in I} d_I(v)$.*

*Proof.* Let $I$ be from Cor A.1. Sample $S \subseteq I$ with $p = \epsilon/2$.

**Size:** $\mathbb{E}[|S|] \geq \epsilon^2 n/6$; Chernoff gives $|S| \geq \epsilon^2 n/12$ w.h.p.

**Decomposition.** $\tilde{L}_S = p^2\tilde{L}_I + p\tilde{W} + \tilde{Q}$ where $\tilde{W} = \sum_v \eta_v\tilde{L}_v^{*,I}$ (linear) and $\tilde{Q} = \sum_e \eta_u\eta_v\tilde{L}_{uv}$ (quadratic), $\eta_v = \xi_v - p$.

**Rank-1 squaring:** $\tilde{L}_{uv}^2 = R_{\text{eff}}(e)\tilde{L}_{uv} \preceq \epsilon\tilde{L}_{uv}$.

**Quadratic term via decoupling [Ver18].** $\mathbb{E}[\|\tilde{Q}\|] \leq C_0\mathbb{E}[\|\tilde{Q}'\|]$ where $\tilde{Q}' = \sum_e \eta_u\eta'_v\tilde{L}_{uv}$. Conditional on $\eta$: $\tilde{Q}' = \sum_v \eta'_v\tilde{M}_v$ with $\tilde{M}_v = \sum_{u \sim v}\eta_u\tilde{L}_{uv}$.

Variance: $\mathbb{E}_\eta[\sum_v \tilde{M}_v^2] \preceq 2p\epsilon\Pi$ (cross terms vanish; diagonal uses $\tilde{L}_{uv}^2 \preceq \epsilon\tilde{L}_{uv}$; double-counting gives $2\tilde{L}_I \preceq 2\Pi$).

Max: $\|\tilde{M}_v\| \leq d_I(v)\epsilon \leq \Delta_I\epsilon$.

Matrix Bernstein: $\mathbb{E}[\|\tilde{Q}\|] \leq O(\sqrt{p\epsilon\log n} + \Delta_I\epsilon\log n)$.

**Linear term:** Similarly, $\mathbb{E}[\|p\tilde{W}\|] \leq O(\sqrt{p^3\Delta_I\epsilon\log n} + p\Delta_I\epsilon\log n)$.

**Total:** $\mathbb{E}[\|\tilde{L}_S\|] \leq \epsilon^2/4 + O(\epsilon\sqrt{\log n}) + O(\Delta_I\epsilon\log n)$.

Markov + Chernoff union bound gives the result. $\square$

## Gaps and Obstacles

### Gap 1: Subset Size

Subset size is $\Theta(\epsilon^2 n)$, not $\Theta(\epsilon n)$. The deterministic term $p^2\|\tilde{L}_I\|$ forces $p = O(\sqrt{\epsilon})$, giving $|S| = O(\epsilon^{3/2}n)$ at best.

### Gap 2: Max Term

The matrix Bernstein max term $\Delta_I\epsilon\log n$ depends on $\Delta_I$, which is unbounded. Removing high-degree vertices requires bounding $|E(I,I)|$, which has no a priori bound.

### Gap 3: Matrix Dimension Penalty

The variance term $\epsilon\sqrt{\log n}$ carries a $\sqrt{\log n}$ penalty from the matrix dimension in matrix Bernstein (Golden-Thompson / Lieb). Generic chaining cannot remove this: intrinsic dimension $d_{\text{eff}} = n - 1$ for worst-case graphs.

### Failed Approaches

| Approach | Obstruction |
|----------|-------------|
| Hanson-Wright + ε-net | Exponent $O(\epsilon)$ vs need $\Omega(n)$ |
| Matrix exponential MCE | Requires $\Delta_I < 0.28/\log n$ |
| MSS / Kadison-Singer | $\delta = \max \ell_v/2 \geq 1$ generically |
| Generic chaining | $d_{\text{eff}} = n-1$, no improvement |
| Leverage separation | Removing $\ell_v > \delta$ costs too many vertices |
| Dim-free tensors [ACSA25, AV25] | Requires i.i.d. summands; our matrices are non-identical |
| Free probability [BvH21, vH24] | **Confirms** $\sqrt{\log n}$ is tight for non-identical sums |
| MSS edge partitioning [PZ23] | Partitions edges, not vertices; opposite goal (sparsification) |
| Matrix pigeonhole on $k$-partition | $\sum_i M_i \preceq \Pi$ does NOT imply $\min_i \|M_i\| \leq 1/k$ |
| Random $k$-partition + Bernstein | Works for $n \leq e^{O(1/\epsilon)}$; $\sqrt{\log n}$ kills it asymptotically |
| Interlacing families (MSS) | Expected char. poly. of $\tilde{L}_{S_j}$ over random $k$-colorings is **not real-rooted** (max |Im| up to 0.34). Tested $K_4$–$K_{12}$, Petersen, cycles. |
| Trace "all bins bad" contradiction | $\sum_j \text{tr}(A_j^{-1}\delta_{v,j}) \geq k$ needed, but sum already exceeds $k$ when bins are NOT bad (11.75 vs $k = 10$ for $K_{20}$). |
| Leverage ordering | Ascending/descending/random: 0/500 stuck for all graphs. Ordering irrelevant for $K_n$ (uniform leverage). |

### Structural Observations (Session 9)

- Expected char. poly. has all roots' real parts ≤ ε despite not being real-rooted — polynomial "knows" the answer
- Cross-bin witness: bad direction of bin $j$ gives ratio ≈0.33 in other bins — **bad directions don't align** (Laplacian structure)
- $K_n$: $\mu_{\max}$ converges to $1/k \approx \epsilon/2$, independent of ordering

### What Would Close the Proof (Updated after Session 11)

**Routes killed:** vanilla interlacing families (not real-rooted); trace-based contradiction (sum exceeds $k$ regardless); $c = 1/2$ via partitions (barbell obstruction); barrier/potential approach (circular); Weaver/KS2 bounds (too loose at $\approx 1.91\epsilon$).

**Routes still alive:**
1. **Minimax-maximin gap $C \leq 3$:** Prove $\min_j \|M_j\| \leq C \cdot \max_w \min_j w^T M_j w$ for graph Laplacian partitions. Empirically $C \leq 2$. Would close $c = 1/3$.
2. **$\min R_{\text{eff}} < \epsilon$ when $b_v = k$:** Verified computationally (0 violations). Structural graph theory argument needed.
3. **Non-real-rootedness polynomial method:** expected polynomial has roots ≤ ε (verified). Probabilistic argument might work without real-rootedness.
4. **Global local-search:** Minimize $\sum_j (-\log\det A_j)$ over partitions via vertex swaps.

## Computational Evidence

We tested the largest ε-light subset across 8 graph families (complete, cycle, random $d$-regular, disjoint cliques, barbell, star, path, Erdős-Rényi) for $n \in \{20, 50, 100\}$ and $\epsilon \in \{0.05, 0.1, 0.2, 0.3, 0.5\}$, using four heuristics (greedy grow, greedy shrink, random sampling, leverage-weighted sampling).

**Result.** In every case tested, $|S|/(\epsilon n) \geq 0.8$. The conjecture appears to be **YES** with $c$ close to 1.

### Multi-Bin Log-Det Barrier

The log-det barrier $\Psi_j = -\log\det(A_j)$ where $A_j = \epsilon\Pi - \tilde{L}_{S_j}$ eliminates the $\delta \preceq A/2$ condition — only $\mu_{\max} < 1$ is needed. Amplification $\leq 1/(1-\mu_{\max})$.

With $k = \lceil 3/\epsilon \rceil$ bins: $\mu_{\max} \leq 2/3$ in best bin for all tested graphs. Margin $\geq 1/3$ uniformly.

### Barbell Obstruction (Proposition G)

Barbell$_{10}$ at $\epsilon = 0.2$, $k = 10$: bridge vertex hits $\mu_{\max} = 1$ exactly. Bridge edge has $R_{\text{eff}} = 1$ (cut edge). Proved: when $m = k = \lceil 2/\epsilon \rceil$, each clique-neighbor bin has $\mu_{\max} = \epsilon/\epsilon = 1$. **$c = 1/2$ via partitions is impossible.**

### Star Norm Identity (Lemma H)

$\|\tilde{L}_v^*\| = 1$ for all vertices $v$ in all connected graphs. Proof: $L_v^* \preceq L$ (PSD complement), equality via isolated-vertex argument. Yields joint bound $\sum_j M_j \preceq \Pi$ and scalar pigeonhole $\min_j w^T M_j w \leq 1/k \leq \epsilon/3$.

### $C$-Sweep Results

| $C$ | Barbell ($m=k+1$) | $K_n$ | All graphs |
|-----|-------------------|-------|------------|
| 2.0 | $\mu \leq 0.91$, STUCK at $m=k$ | $\leq 0.83$ | Fails |
| 2.5 | $\mu \leq 0.74$ | $\leq 0.74$ | 0 stuck |
| 3.0 | $\mu \leq 0.64$ | $\leq 0.67$ | 0 stuck, margin $\geq 1/3$ |
| 4.0 | $\mu = 0$ | $\leq 0.56$ | 0 stuck, margin $\geq 0.44$ |

**Most likely proof direction.** Prove the minimax-maximin gap $C \leq 3$ for graph Laplacian partitions, converting the scalar pigeonhole ($\epsilon/3$) to a spectral norm bound ($\epsilon$). Empirically $C \leq 2$ for all tested graphs.
